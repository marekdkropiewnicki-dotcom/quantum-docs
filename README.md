## ✦ Security as Symmetry

- Two-factor authentication is not optional — it is the **conservation law** of identity.
- Dependabot, secret scanning, and code scanning run as background radiation.
- Least privilege. Always. For tokens, for keys, for trust itself.
- Found a vulnerability? Whisper it privately to the enterprise owner before the world hears.
